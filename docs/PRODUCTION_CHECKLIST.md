# Production Checklist

Before using the plugin on a live site, confirm:

* Workflow Active
* Production URL saved
* Allowed Origins configured
* Memory connected when previous sessions are enabled
* Plugin enabled
* Device visibility set as intended
* Page visibility set as intended
* Quick actions tested
* Dynamic options tested
* Email follow-up workflow tested in n8n
* Mobile tested
* Cache purged
* Privacy policy updated

Also verify the plugin dashboard shows the current setup status, display mode, and render target you expect.
