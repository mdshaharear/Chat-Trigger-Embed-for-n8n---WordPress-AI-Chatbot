# Quick Start

1. Upload the plugin ZIP in WordPress.
2. Activate the plugin.
3. Open `Chat Trigger Embed` and go to `Connection`.
4. Configure the n8n Chat Trigger node.
5. Add your website origin to `Allowed Origins`.
6. Paste the production Chat Trigger URL.
7. Connect Redis memory or another supported memory node if you want previous sessions.
8. Enable the chatbot in the plugin.
9. If you use Elementor, drag the `AI Chat Builder` widget into the page or template.
10. Customize appearance and quick actions.
11. Clear site cache and test in an incognito browser.

Keep the browser-facing Chat Trigger URL public. Do not store secrets in plugin settings.
