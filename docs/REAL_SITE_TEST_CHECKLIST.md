# Real Site Test Checklist

1. Verify you have a backup.
2. Upload the plugin ZIP through `Plugins -> Add New -> Upload Plugin`.
3. Activate the plugin.
4. Open every admin page.
5. Save a harmless settings change.
6. Create a test profile.
7. Open Runtime Lab.
8. Run mock chat tests.
9. Configure the production n8n URL.
10. Add the allowed origins in n8n.
11. Activate the n8n workflow.
12. Run the live test with confirmation.
13. Check the frontend on desktop, tablet, and mobile.
14. Test quick actions, dynamic options, and pre-chat form behavior.
15. Reload the page and confirm session restore behavior.
16. Purge any cache plugin or host cache.
17. Review the PHP log and browser console.
18. Export the sanitized runtime report.
